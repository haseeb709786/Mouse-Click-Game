2D game built with Java Processing. 
The player controls a blue ship (mouse-controlled) and must protect a stationary goal in the center of the screen from incoming enemies. 
Enemies come in different types: normal red rockets, stronger blue planets, and hovering mines that damage the player when touched.

Gameplay
Click on enemies to destroy them before they reach the goal.
Hovering over HoverEnemies (yellow mines) costs lives.
StrongEnemies require more clicks or deal more damage if they reach the goal.
The game has multiple levels as you score points, new levels spawn with more frequent and tougher enemies.
The game ends when lives reach zero.
